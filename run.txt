### 4. Create a PBS job script for the full runs

After verifying the exact command, create a PBS script that runs one experiment for one diversity target.

Template:

```bash
cat > run_divclust_dt.pbs <<'EOF'
#!/bin/bash
#PBS -N divclust_dt
#PBS -l select=1:ncpus=4:mem=16gb
#PBS -l walltime=08:00:00

cd $PBS_O_WORKDIR
mkdir -p logs results

module load anaconda3
source $(conda info --base)/etc/profile.d/conda.sh
conda activate divclust

DT_SAFE=$(echo ${DT} | tr '.' '_')

echo "Started DT=${DT} at $(date)"

python main.py --preset cc_cifar10 --epochs 100 <diversity-flag> ${DT} \
  > logs/divclust_dt_${DT_SAFE}.out 2>&1

echo "Finished DT=${DT} at $(date)"
EOF
```

Replace `<diversity-flag>` with the actual discovered flag or config override syntax.

### 5. Submit the 5 table runs

Submit one job for each diversity target:

```bash
for DT in 1.0 0.95 0.9 0.8 0.7; do
  qsub -v DT=$DT run_divclust_dt.pbs
done
```

These should be submitted separately so they can run independently.
