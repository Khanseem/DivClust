Instruction for Codex: Run Smoke Test for DivClust

Your task is to verify that the DivClust training pipeline runs correctly on the NUS HPC environment before launching the full experiments.

Follow these steps exactly.

1. Navigate to the repository
cd ~/DivClust

If the repo does not exist yet:

git clone https://github.com/Khanseem/DivClust.git
cd DivClust
2. Activate the environment
module load anaconda3
source $(conda info --base)/etc/profile.d/conda.sh
conda activate divclust

If the environment does not exist yet:

conda create -n divclust python=3.10 -y
conda activate divclust
pip install torch torchvision scipy wandb PyYAML scikit-learn termcolor matplotlib opencv-contrib-python
3. Run a smoke test (VERY SHORT RUN)

Run only 2 epochs with a single clustering to confirm everything works.

python main.py \
  --preset cc_cifar10 \
  --clusterings 2 \
  --epochs 2 \
  --NMI_target 0.8

Explanation:

preset cc_cifar10 → uses CIFAR-10 CC framework config

clusterings 2 → very small ensemble

epochs 2 → quick test

NMI_target 0.8 → example diversity target
